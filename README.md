# بصيرة – Basira

## الوصف
بصيرة هو نظام مساعد ذكي للمكفوفين يعتمد على الذكاء الاصطناعي وتقنيات الرؤية الحاسوبية.  
يهدف إلى دعم استقلالية المستخدم من خلال التعرف على الأشياء، قراءة النصوص (OCR)، التعرف على الأغراض الشخصية، وقراءة الوقت، مع إخراج النتائج صوتيًا باللغة العربية عبر سماعة الأذن.

---

## طريقة التشغيل

```bash
pip install torch torchvision opencv-python easyocr numpy vosk pyaudio gtts playsound keyboard openai sounddevice
```


## التقنيات
- Python  
- OpenCV (معالجة الصور)  
- PyTorch (نماذج الذكاء الاصطناعي)  
- YOLOv5 (كشف الأشياء)  
- MobileNetV2 (التعرف على العناصر الشخصية)  
- EasyOCR (قراءة النصوص)  
- Vosk (التعرف على الصوت)  
- SQLite (تخزين البيانات)  
- elm-TTS / gTTS (تحويل النص إلى صوت)
